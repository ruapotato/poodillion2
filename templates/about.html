Poodillion is a web-based Unix hacking simulator set in the early 1990s. It provides a complete virtual Unix environment where you can learn hacking, networking, and system administration in a safe, legal environment.

üéÆ Features

  • Complete Unix Emulation - Virtual filesystem, processes, permissions
  • Virtual Network - Multiple systems, routing, firewalls
  • PooScript Language - Hackable scripting language for all binaries
  • Realistic Commands - ls, ps, nmap, ssh, and more
  • Educational - Learn real Unix and networking concepts
  • 100% Browser-Based - No installation required

üöÄ Getting Started

Start by running basic commands:

  • help - Show available commands
  • ls - List files
  • cat notes.txt - Read files
  • nmap 192.168.1.0/24 - Scan the network
  • ps -ef - List processes
  • ifconfig - Show network interfaces

üí° Learning Path

  1. Explore - Use ls, cat, pwd to understand the filesystem
  2. Discover - Find hidden files and clues (find / -name "*.txt")
  3. Scan - Map the network with nmap
  4. Exploit - Find vulnerabilities and gain access
  5. Learn - Read PooScript binaries to understand how everything works

üìù About PooScript

PooScript is the heart of Poodillion. Every command, daemon, and service is written in PooScript - a safe Python-like language. This means:

  • You can read any binary: cat /bin/ls
  • You can modify system behavior
  • You can create your own tools and exploits
  • Nothing is hidden - it's all transparent!

  • Teaching cybersecurity concepts
  • Having fun with retro computing!
